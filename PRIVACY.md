# Privacy Policy for Umbraco Power Tools

**Last Updated:** December 4, 2024

## Introduction

Umbraco Power Tools ("the Extension") is committed to protecting your privacy. This Privacy Policy explains how the Extension collects, uses, and safeguards your information.

## Information We Collect

### User Preferences (Stored Locally)

The Extension stores the following preferences locally on your device using browser storage:

- **Theme Mode**: Your preferred color theme (Light, Dark, or Automatic)
- **Navigation Preference**: Whether links should open in a new tab or current tab

**Important:** These preferences are stored locally in your browser only and are never transmitted to any external servers.

### Current Tab Information

The Extension accesses your current tab's URL only when you click the navigation buttons to:
- Determine if you're already on the `/umbraco` admin path
- Navigate between the Umbraco admin panel and site root

**Important:** Tab URLs are read in real-time and are never stored, logged, or transmitted anywhere.

### Cached Content

The Extension temporarily caches the following content locally for 5 minutes to reduce API requests:
- Latest forum posts from forum.umbraco.com
- Latest Mastodon posts with #Umbraco hashtag

**Important:** This cache is stored locally in your browser only and contains only public information.

## Information We DO NOT Collect

The Extension does NOT:
- Collect any personally identifiable information (PII)
- Track your browsing history
- Use cookies or tracking pixels
- Collect analytics or usage statistics
- Store passwords or authentication credentials
- Access or modify website content
- Share any data with third parties for advertising or marketing purposes

## Third-Party Services

The Extension retrieves publicly available content from the following sources:

### Umbraco Forum (forum.umbraco.com)
- **Purpose**: Display latest forum posts
- **Data Retrieved**: Public forum post titles, reply counts, and timestamps
- **Privacy Policy**: [Umbraco Forum Privacy Policy](https://umbraco.com/legal/privacy-policy/)

### Mastodon (mastodon.social)
- **Purpose**: Display latest posts with #Umbraco hashtag
- **Data Retrieved**: Public post content, usernames, and timestamps
- **Privacy Policy**: [Mastodon.social](https://mastodon.social/about)

**Important:** These requests retrieve only publicly available information. No authentication or personal data is sent to these services.

## Permissions Explanation

The Extension requires the following browser permissions:

- **`storage`**: To save your theme and navigation preferences locally
- **`activeTab`**: To read the current tab's URL for navigation features
- **`tabs`**: To create new tabs or navigate existing tabs
- **`<all_urls>`**: To enable navigation features on any website you're working with

These permissions are used solely for the Extension's core functionality and not for data collection.

## Data Security

- All user preferences are stored locally using the browser's secure storage API
- No data is transmitted to external servers except for retrieving public forum and social media content
- The Extension does not create or use any external databases
- No user data leaves your device

## Data Retention

- **User Preferences**: Stored indefinitely in your browser until you uninstall the Extension or clear browser data
- **Cached Content**: Automatically expires after 5 minutes
- **No Server-Side Storage**: We do not maintain any servers or databases that store user information

## Your Rights

You have the right to:
- **Access**: View your stored preferences in the Extension settings
- **Modify**: Change your preferences at any time through the settings page
- **Delete**: Uninstall the Extension to remove all locally stored data
- **Export**: Browser settings allow you to export extension data

## Children's Privacy

The Extension does not knowingly collect information from children under 13 years of age. The Extension is designed for web developers working with Umbraco CMS.
